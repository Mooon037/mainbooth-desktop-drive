주요 기능

음악 제작 워크플로우를 위해 특별히 설계된 기능들

OS 네이티브 통합

macOS Finder와 Windows Explorer에 완벽하게 통합되어 일반 폴더처럼 사용할 수 있습니다.

실시간 동기화

파일 변경사항이 실시간으로 동기화되어 팀원들과 즉시 공유할 수 있습니다.

스마트 캐싱

자주 사용하는 파일은 로컬에 캐시되어 빠른 접근이 가능합니다.

버전 관리

파일의 모든 버전이 자동으로 저장되어 언제든지 이전 버전으로 되돌릴 수 있습니다.

권한 관리

프로젝트별로 세밀한 권한 설정을 통해 안전한 협업 환경을 제공합니다.

모바일 연동

Main Booth 모바일 앱과 완벽하게 연동되어 어디서나 프로젝트에 접근할 수 있습니다.

설치 가이드

간단한 몇 단계로 Main Booth Drive를 설치하세요

1

DMG 파일 다운로드

위의 다운로드 버튼을 클릭하여 DMG 파일을 다운로드합니다.

2

DMG 파일 실행

다운로드된 DMG 파일을 더블클릭하여 마운트합니다.

3

Applications 폴더로 드래그

Main Booth Drive 앱을 Applications 폴더로 드래그하여 설치합니다.

4

앱 실행 및 로그인

Applications 폴더에서 앱을 실행하고 Main Booth 계정으로 로그인합니다.

1

EXE 파일 다운로드

위의 다운로드 버튼을 클릭하여 설치 프로그램을 다운로드합니다.

2

설치 프로그램 실행

다운로드된 EXE 파일을 우클릭하여 "관리자 권한으로 실행"을 선택합니다.

3

설치 과정 진행

설치 마법사의 안내에 따라 설치를 완료합니다.

4

앱 실행 및 로그인

바탕화면 아이콘을 더블클릭하여 앱을 실행하고 로그인합니다.

1

AppImage 파일 다운로드

위의 다운로드 버튼을 클릭하여 AppImage 파일을 다운로드합니다.

2

실행 권한 부여

터미널에서 chmod +x MainBoothDrive.AppImage 명령을 실행합니다.

3

FUSE 설치 (필요시)

Ubuntu: sudo apt install fuse
CentOS: sudo yum install fuse

4

앱 실행

AppImage 파일을 더블클릭하거나 터미널에서 실행하여 로그인합니다.
